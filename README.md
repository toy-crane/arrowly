# Arrowly

**화면 위에 바로 긋는 형광펜**

녹화 중인 화면에 밑줄·동그라미·화살표를 얹고, 설명이 끝나면 흔적 없이 사라집니다.
화면 녹화 강의를 만드는 사람을 위한 일회용 주석 오버레이입니다.

## 30초 시작

1. 위의 DMG를 받아 열고, **Arrowly를 Applications 폴더로** 끌어넣습니다
2. 실행합니다 — 베타 빌드라 경고가 뜨면 [여기](#처음-열-때-경고가-떠요)를 보세요
3. 첫 실행이면 20초 온보딩이 뜹니다. 끝나면 키 4개가 전부입니다:

| 키 | 동작 |
|---|---|
| **⌥ Tab** | 그리기 켜기 / 끄기 |
| **⌘ Z** | 마지막 획 취소 (⇧⌘Z로 되살리기) |
| **⌥ ⌫** | 전체 지우기 |
| **Esc** | 언제나 그리기에서 빠져나오기 |

그리기를 켜면 화면 왼쪽 아래에 작은 **마커 캡슐**이 떠 있습니다 — 색 5가지, 굵기 5단계를 그 자리에서 바꾸고, 드래그로 옮길 수 있습니다.

## 왜 Arrowly인가

**녹화 화면이 죽어 보이지 않습니다.**
그리는 동안에도 아래 앱의 메뉴바와 창이 활성 상태 그대로 유지됩니다. ⌘Z까지 오버레이가 받아 처리하면서도요. 시청자는 오버레이가 떠 있다는 사실조차 모릅니다.

**권한 0개.**
손쉬운 사용(접근성) 권한 없이 동작하도록 처음부터 설계했습니다. 시스템 설정을 뒤질 일 없이, 설치하면 바로 씁니다.

**일회용 주석이라는 철학.**
획은 선택되지도, 이동되지도, 저장되지도 않습니다. 말하는 동안만 존재하는 형광펜 자국입니다. 화면을 움직이기 전에 ⌥⌫ 한 번 — 그게 관리의 전부입니다. 다이어그램 편집기가 필요하면 Excalidraw를 쓰세요. Arrowly는 그 선을 넘지 않습니다.

## 설치

### 처음 열 때 경고가 떠요

오픈 베타는 아직 **서명되지 않은 빌드**라 macOS가 첫 실행을 막습니다. 정식 배포 전까지는 아래 방법으로 한 번만 열어주면 됩니다 (이후엔 묻지 않습니다):

- **macOS 15 (Sequoia) 이상**: 앱을 한 번 실행해 경고를 닫은 뒤 → **시스템 설정 → 개인정보 보호 및 보안** → 맨 아래 "Arrowly이(가) 차단되었습니다" 옆의 **그래도 열기**
- **macOS 14 이하**: Applications 폴더에서 Arrowly를 **우클릭(⌃-클릭) → 열기 → 열기**

> 서명·공증된 빌드는 준비 중입니다. 적용되면 이 절차는 사라집니다.

### 사용 팁

- **그림이 사라졌어요?** — 통과 모드로 나가면 그림은 *숨김*일 뿐 삭제되지 않습니다. 다시 ⌥Tab을 누르면 그대로 돌아옵니다. 삭제는 ⌘Z와 ⌥⌫뿐입니다.
- **다른 모니터에 그리고 싶어요** — 그 모니터로 커서를 옮기고 ⌥Tab을 누르면 됩니다 (모니터가 바뀌면 이전 그림은 비워집니다).
- **⌥Tab이 다른 앱과 겹쳐요** — 메뉴바의 화살표 아이콘 → **단축키 설정…** 에서 언제든 바꿀 수 있습니다.
- **깨끗한 테이크가 필요해요** — 메뉴바 아이콘 → **마커 숨기기**를 켜면 마커 없이 그릴 수 있습니다.
- **튜토리얼 다시 보기** — 메뉴바 아이콘 메뉴에 있습니다.

## 오픈 베타 안내

지금은 실사용 피드백을 모으는 단계입니다. 알려진 범위와 제약:

- **macOS 전용**입니다 (Windows는 오버레이 코어를 다시 설계해야 해서 별도 트랙입니다)
- 오버레이는 **커서가 있는 모니터 1개**만 덮습니다
- 그림은 **정지된 화면 기준**입니다 — 아래 화면을 스크롤해도 따라가지 않습니다 (설계상 의도)
- 자동 업데이트가 없어 새 버전은 재다운로드해야 합니다

버그·아이디어·거슬리는 점 전부 환영합니다 → [GitHub Issues](https://github.com/toy-crane/arrowly/issues)

## FAQ

**왜 접근성 권한이 필요 없나요?**
키를 가로채는 이벤트 탭 대신, 포커스를 훔치지 않는 패널(NSPanel) 구조로 키 입력을 받습니다. 권한이 필요 없고, 그래서 아래 앱이 활성으로 유지됩니다.

**App Store에는 없나요?**
투명 오버레이 구조가 App Store 정책과 맞지 않아 v1은 DMG 직접 배포입니다.

**어떤 Mac에서 돌아가나요?**
Apple Silicon과 Intel 모두 지원하는 universal 빌드입니다.

## 개발자용

```bash
bun install
bun tauri dev        # 개발 실행
bun tauri build --debug --bundles app   # 로컬 무서명 빌드
```

- 스택: Tauri v2 + React/TypeScript + Bun (요구사항: [docs/REQUIREMENTS.md](docs/REQUIREMENTS.md))
- 릴리스 절차: [docs/RELEASE.md](docs/RELEASE.md) · 아이콘 재생성: `scripts/gen-icons.sh`
