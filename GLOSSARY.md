# Glossary

Arrowly에서 화면 위에 즉석으로 남기는 판서 요소와 국소 교정에 사용하는 용어를 정의한다.

**마크 (Mark)**:
현재 오버레이에 확정되어 전체 지우기와 실행 취소의 한 단위로 취급되는 판서 요소.

**텍스트 마크 (Text mark)**:
타이핑으로 만든 마크. 다른 마크와 달리 내용·텍스트 크기·위치를 교정할 수 있다.
_피하기_: 텍스트 객체, 텍스트 박스

**텍스트 초안 (Text draft)**:
새 텍스트 마크로 확정되기 전 입력 중인 텍스트.

**텍스트 편집 세션 (Text editing session)**:
텍스트 초안을 작성하거나 기존 텍스트 마크를 다시 열어 내용과 크기를 바꾸는 일시적인 상태.
_피하기_: 선택 모드, 객체 편집 모드

**텍스트 이동 제스처 (Text move gesture)**:
텍스트 모드에서 기존 텍스트 마크 하나를 끌어 위치만 교정하는 일회성 동작. 선택 상태를 만들지 않는다.

**텍스트 크기 (Text size)**:
펜 굵기와 독립적으로 텍스트 마크에 적용되는 5단계 표시 크기.

**직선 잠금 제스처 (Line lock gesture)**:
열린 자유곡선의 끝을 잠시 유지해 직선으로 보정한 뒤, 놓기 전까지 끝점을 조절하는 일회성 동작. 선택 상태나 이후 편집 상태를 만들지 않는다.
