# Glossary

Arrowly에서 화면 위에 즉석으로 남기는 판서 요소, 도구, 상태의 표준 용어를 정의한다.

## 판서 요소

**주석 오버레이 (Annotation overlay)**:
다른 앱의 정지된 화면 위를 덮어 즉석 판서를 제공하는 Arrowly의 작업 표면.
_피하기_: 문서, 캔버스 파일

**마크 (Mark)**:
주석 오버레이에 확정된 하나의 판서 요소.
_피하기_: 객체, 레이어

**펜 마크 (Pen mark)**:
자유곡선 도구로 그린 경로가 확정된 마크.
_피하기_: 스트로크 객체

**직선 마크 (Line mark)**:
직선 또는 화살표 도구로 만든 열린 기하 마크. 끝점 화살촉을 가질 수 있다.
_피하기_: 도형 마크, 연결선

**도형 마크 (Shape mark)**:
사각형, 타원, 삼각형처럼 닫힌 기하 형태로 확정된 마크.
_피하기_: 편집 가능한 도형 객체

**텍스트 마크 (Text mark)**:
타이핑한 글자가 확정된 마크.
_피하기_: 텍스트 객체, 텍스트 박스

**진행 중 획 (Live stroke)**:
포인터를 누른 뒤 놓기 전까지 아직 마크로 확정되지 않은 자유곡선.
_피하기_: 마크

## 작업 상태와 도구

**그리기 모드 (Drawing mode)**:
주석 오버레이가 보이고 포인터 입력을 받아 판서할 수 있는 앱 상태.

**통과 모드 (Pass-through mode)**:
주석 오버레이가 숨고 포인터 입력이 아래 앱으로 전달되는 앱 상태.
_피하기_: 그리기 OFF

**이동 탐색 상태 (Move discovery state)**:
`⌘`을 단독으로 잠시 누르는 동안 이동할 수 있는 모든 마크를 일시적으로 드러내는 상태. 지속적인 선택을 만들지 않는다.
_피하기_: 선택 모드, 이동 모드

**삭제 탐색 상태 (Deletion discovery state)**:
`⌥`을 단독으로 잠시 누르는 동안 삭제할 수 있는 모든 마크를 일시적으로 드러내는 상태. 지속적인 선택을 만들지 않는다.
_피하기_: 선택 모드, 지우개 크기

**블랙보드 (Blackboard)**:
그리기 모드에서 투명한 화면 배경을 대신하는 불투명 검정 배경 상태.
_피하기_: 블랙보드 모드, 보드 모드

**자유곡선 도구 (Freehand tool)**:
진행 중 획을 시작해 펜 마크를 만드는 판서 도구. 획의 모양이나 정지 시간으로 다른 마크를 자동 생성하지 않는다.
_피하기_: 펜 모드

**텍스트 도구 (Text tool)**:
텍스트 초안의 위치를 정하거나 기존 텍스트 마크를 교정하는 판서 도구.
_피하기_: 텍스트 모드, T 모드

**마크 삭제 도구 (Mark deletion tool)**:
마크 하나를 맞혀 전체를 제거하고, 다른 도구를 고를 때까지 활성 상태를 유지하는 판서 도구.
_피하기_: 부분 지우개, 픽셀 지우개

**기하 그리기 도구 (Geometric drawing tool)**:
직선, 끝점 화살표, 사각형, 타원 또는 삼각형을 고른 뒤 다른 도구를 고를 때까지 같은 형태의 마크를 반복해서 만드는 판서 도구.
_피하기_: 빠른 추가, 일회성 도형, 도형 편집 모드

**플로팅 마커 (Floating marker)**:
현재 도구와 잉크 속성을 보여주고 바꾸는 주석 오버레이 위의 작은 컨트롤.
_피하기_: 툴바, 도구 막대

**도구 속성 패널 (Tool inspector)**:
플로팅 마커에서 현재 도구에 해당하는 속성만 펼쳐 보여주는 패널.
_피하기_: 팝오버 메뉴

## 텍스트 교정

**텍스트 초안 (Text draft)**:
새 텍스트 마크로 확정되기 전 입력 중인 텍스트.

**텍스트 편집 세션 (Text editing session)**:
텍스트 초안을 작성하거나 기존 텍스트 마크의 내용과 크기를 교정하는 일시적인 상호작용.
_피하기_: 선택 모드, 객체 편집 모드

**텍스트 크기 (Text size)**:
펜 굵기와 독립적으로 텍스트 마크에 적용되는 표시 크기 단계.

## 마크 이동

**마크 이동 제스처 (Mark move gesture)**:
이동 탐색 상태에서 확정된 마크 하나를 끌어 위치만 교정하는 일회성 동작.

## 임시 강조

**포인터 핑 (Pointer ping)**:
빈 곳이나 텍스트가 아닌 마크를 더블클릭한 위치를 짧은 파티클 버스트로 알리는 일시적 강조. 마크나 이력으로 남지 않는다.
_피하기_: 커서 하이라이트, 스포트라이트 마크
