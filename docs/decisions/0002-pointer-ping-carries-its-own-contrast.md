# 포인터 핑은 스스로 대비를 갖는다

포인터 핑은 핑크 `#FF2D95`에 어두운 대비 외곽선을 둘러 배경과 무관하게 읽히도록 한다. 확정된 마크에는 [외곽선을 두지 않고](../specs/drawing/spec.md), 밝은 배경에서 색이 약하면 사용자가 `⌘1`–`⌘5`로 다른 색을 고르게 한다. 핑에는 그 탈출구가 없다. 핑의 색은 [설정 대상이 아니고](../specs/quick-mark-tools/spec.md) 오버레이가 덮는 화면도 고를 수 없으므로, 색 하나로는 어떤 배경에서도 보인다고 보장할 수 없다. 옐로우 `#FFD400`은 흰 배경에서 대비 1.4:1로 사실상 보이지 않았고, 핑크로 바꿔도 3.3:1에 그친다. 마크와 핑에 같은 외곽선 규칙을 적용하는 일관성보다, 사용자가 손댈 수 없는 요소가 항상 보이는 쪽을 택한다.
