# 그리기와 마크 이력

## 목적

도구 전환 없이 자유곡선으로 빠르게 설명하고, 필요한 경우 같은 제스처 안에서 닫힌 도형이나 직선으로 보정한다. 확정된 판서는 마크 단위로 남고 즉시 실행 취소할 수 있다.

## 자유곡선 도구

- 자유곡선 도구 하나로 밑줄, 원, 손그림 화살표를 포함한 펜 마크를 만든다.
- 홀드 전에 놓은 진행 중 획은 모양을 해석하지 않고 펜 마크로 확정한다.
- 홀드 보정은 닫힌 획을 사각형·타원으로, 열린 획을 직선으로 확정할 수 있다. 직선 세부 동작은 [직선 잠금 제스처](../line-lock-gesture/spec.md)를 따른다.
- 별도 도형·직선·화살표 도구는 제공하지 않는다.

## 잉크 속성

- 색은 옐로우 `#FFD400`, 오렌지 `#FF7A00`, 핑크 `#FF2D95`, 그린 `#2ED573`, 블루 `#00AEEF` 다섯 가지다.
- 펜 굵기는 다섯 단계이며 기본값은 가운데 단계다.
- 획에는 대비 외곽선을 두지 않는다. 밝은 배경에서 옐로우가 약하면 사용자가 다른 색을 선택한다.
- 색과 굵기는 진행 중 획을 시작할 때 결정하며 그리는 도중 바꾸지 않는다.
- 현재 색과 굵기는 점 커서와 [플로팅 마커](../floating-marker-control/spec.md)에 표시한다.

## 마크와 이력

- 진행 중 획은 포인터를 놓을 때 펜 마크 또는 도형 마크 하나로 확정된다.
- 텍스트 마크를 포함한 모든 마크는 전체 지우기의 대상이다.
- 새 마크 추가, 텍스트 마크 교체·이동·삭제는 각각 실행 취소 한 단위다.
- ⌘Z는 마지막 이력을 취소하고 ⇧⌘Z는 되살린다.
- 새 변경이 생기면 취소 후 남아 있던 다시 실행 이력은 폐기한다.
- 전체 지우기는 모든 마크와 진행 중 획을 비우는 1급 동작이다.

## 렌더링 품질

- Retina 화면에서도 펜 마크가 흐리지 않아야 한다.
- 빠른 포인터 이동에서도 경로가 눈에 띄게 끊기지 않아야 한다.
- 화면 녹화 중 입력 지연이 강의 흐름을 방해하지 않아야 한다.
- 그리기 모드를 나가거나 포인터 입력이 취소되면 진행 중 획과 보정 미리보기를 남기지 않는다.

## 인수 조건

- 한 번의 포인터 제스처가 정확히 마크 하나를 만든다.
- 점 클릭, 짧은 경로, 빠른 경로가 모두 보이는 펜 마크로 확정된다.
- 마크 추가·교체·이동·삭제의 Undo/Redo가 원래 z-order를 보존한다.
- 실행 취소 뒤 새 마크를 만들면 이전 Redo가 다시 나타나지 않는다.
- 전체 지우기와 통과 모드 전환 중 진행 중 획이 남지 않는다.
- 다섯 색과 다섯 굵기가 커서, 플로팅 마커, 확정된 펜 마크에서 일치한다.

## 비범위

- 부분 지우개
- 확정된 펜 마크나 도형 마크의 선택·이동·교정
- 화살표 머리 인식과 자동 생성
- 화면 내용이나 다른 마크에 대한 스냅·정렬
