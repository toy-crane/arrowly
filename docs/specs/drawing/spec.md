# 그리기와 마크 이력

## 목적

자유곡선으로 빠르게 설명하고, 필요할 때 명시적인 기하 그리기 도구로 또렷한 화살표·도형을 반복해서 만든다. 확정된 판서는 마크 단위로 남고 추가·이동·교정·삭제를 즉시 실행 취소할 수 있다.

## 자유곡선과 기하 그리기

- 자유곡선 도구는 `350ms` 전에 놓으면 포인터 경로를 펜 마크로 확정한다.
- 자유곡선 끝에서 오래 멈추면 `150ms`부터 진행 링을 표시하고 `350ms`에 열린 획은 직선, 닫힌 획은 사각형·타원·삼각형으로 보정한다.
- 끝점 화살표·사각형·타원·삼각형은 도구 속성 패널에서 명시적으로 선택한다.
- 별도 직선 도구는 제공하지 않는다. 화살촉 없는 직선은 자유곡선 끝에서 오래 눌러 만든다.
- 기하 그리기 도구는 다른 도구를 고를 때까지 유지되어 같은 형태의 마크를 연속 생성한다.
- 도구 선택, 지속 상태, 플로팅 마커 표시와 제스처 계약은 [마크 도구 선택·이동·삭제 일관성](../mark-tool-consistency/spec.md)을 따른다.

## 잉크 속성

- 색은 옐로우 `#FFD400`, 오렌지 `#FF7A00`, 핑크 `#FF2D95`, 그린 `#2ED573`, 블루 `#00AEEF` 다섯 가지다.
- 펜 굵기는 다섯 단계이며 기본값은 가운데 단계다.
- 펜 굵기는 펜 마크, 직선 마크, 도형 마크와 화살촉에 함께 적용된다.
- `⌘+`와 `⌘−`는 자유곡선 또는 기하 그리기 도구가 활성일 때 굵기를 한 단계씩 바꾼다.
- 획에는 대비 외곽선을 두지 않는다. 밝은 배경에서 옐로우가 약하면 사용자가 다른 색을 선택한다.
- 색과 굵기는 포인터 제스처를 시작할 때 결정하며 그리는 도중 바꾸지 않는다.
- 현재 색과 굵기는 점 커서와 [플로팅 마커](../floating-marker-control/spec.md)에 표시한다.

## 마크와 이력

- 한 번의 포인터 제스처는 펜 마크, 직선 마크 또는 도형 마크 하나를 확정한다.
- 텍스트를 포함한 모든 확정 마크는 이동과 삭제의 대상이다.
- 새 마크 추가, 마크 이동, 텍스트 교정과 마크 삭제는 각각 실행 취소 한 단위다.
- 이동·삭제의 표적화와 상태 전이는 [마크 도구 선택·이동·삭제 일관성](../mark-tool-consistency/spec.md)을 따른다.
- `⌘Z`는 마지막 이력을 취소하고 `⇧⌘Z`는 되살린다. 활성 그리기 도구는 바뀌지 않는다.
- 새 변경이 생기면 취소 후 남아 있던 다시 실행 이력은 폐기한다.
- 전체 지우기는 모든 마크와 진행 중 입력을 비우되 현재 선택한 그리기 도구를 유지한다.

## 렌더링 품질

- Retina 화면에서도 모든 마크와 화살촉이 흐리지 않아야 한다.
- 빠른 포인터 이동에서도 경로가 눈에 띄게 끊기지 않아야 한다.
- 화면 녹화 중 입력 지연이 강의 흐름을 방해하지 않아야 한다.
- 그리기 모드를 나가거나 포인터 입력이 취소되면 진행 중 획과 기하 미리보기를 남기지 않는다.

## 인수 조건

- 점 클릭, 짧은 경로와 빠른 경로가 모두 보이는 펜 마크로 확정된다.
- 자유곡선을 오래 누르면 직선·사각형·타원·삼각형 미리보기가 나타나고 놓을 때 실행 취소 가능한 마크 하나로 확정된다.
- 자유곡선을 보정 전에 놓거나 너무 작거나 퇴화한 획으로 그리면 펜 마크로 남는다.
- 명시적인 기하 그리기 도구로 화살표·사각형·타원·삼각형을 각각 연속 생성할 수 있다.
- Shift 유무와 관계없이 화살표의 끝점이 실제 포인터 위치와 일치한다.
- 모든 확정 마크의 이동·삭제와 그 Undo/Redo가 위치와 원래 z-order를 보존한다.
- 실행 취소 뒤 새 마크를 만들면 이전 Redo가 다시 나타나지 않는다.
- 전체 지우기와 통과 모드 전환 중 진행 중 입력이 남지 않는다.
- 다섯 색과 다섯 굵기가 커서, 플로팅 마커와 확정된 잉크 마크에서 일치한다.

## 비범위

- 자유곡선에서 화살표 자동 인식 또는 생성
- 화살표의 방향 제한, 끝점·화살촉 교정
- 펜 마크의 경로 교정과 부분 지우기
- 여러 마크 동시 선택·이동, 그룹, 정렬과 스냅
