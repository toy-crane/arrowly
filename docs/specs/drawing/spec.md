# 그리기와 마크 이력

## 목적

자유곡선으로 빠르게 설명하고, 필요할 때 한 번의 빠른 추가나 같은 자유곡선 제스처의 홀드 보정으로 또렷한 기하 마크를 만든다. 확정된 판서는 마크 단위로 남고 추가·이동·교정·삭제를 즉시 실행 취소할 수 있다.

## 자유곡선과 홀드 보정

- 자유곡선 도구로 밑줄, 원, 손그림 화살표를 포함한 펜 마크를 만든다.
- 홀드 전에 놓은 진행 중 획은 모양을 해석하지 않고 펜 마크로 확정한다.
- 끝점을 `350ms` 유지한 획은 사각형·타원·삼각형·직선으로 보정할 수 있다. 분류·미리보기·취소 동작은 [홀드 보정 제스처](../line-lock-gesture/spec.md)를 따른다.
- 홀드 보정은 화살표를 자동 생성하지 않는다.

## 빠른 추가

- 자유곡선 도구의 속성 패널에서 사각형·타원·삼각형·직선·끝점 화살표를 골라 한 번의 드래그로 정확한 마크를 추가할 수 있다.
- 빠른 추가는 마크 하나를 성공적으로 확정한 뒤 자유곡선 도구로 복귀한다.
- 빠른 추가의 도구 상태·제스처·취소 계약은 [빠른 마크 도구](../quick-mark-tools/spec.md)를 따른다.

## 잉크 속성

- 색은 옐로우 `#FFD400`, 오렌지 `#FF7A00`, 핑크 `#FF2D95`, 그린 `#2ED573`, 블루 `#00AEEF` 다섯 가지다.
- 펜 굵기는 다섯 단계이며 기본값은 가운데 단계다.
- 펜 굵기는 펜 마크, 직선 마크, 도형 마크와 화살촉에 함께 적용된다.
- `⌘+`와 `⌘−`는 자유곡선 또는 빠른 추가가 활성일 때 굵기를 한 단계씩 바꾼다.
- 획에는 대비 외곽선을 두지 않는다. 밝은 배경에서 옐로우가 약하면 사용자가 다른 색을 선택한다.
- 색과 굵기는 포인터 제스처를 시작할 때 결정하며 그리는 도중 바꾸지 않는다.
- 현재 색과 굵기는 점 커서와 [플로팅 마커](../floating-marker-control/spec.md)에 표시한다.

## 마크와 이력

- 진행 중 획은 포인터를 놓을 때 펜 마크, 직선 마크 또는 도형 마크 하나로 확정된다.
- 텍스트 마크를 포함한 모든 마크는 [마크 삭제 도구](../quick-mark-tools/spec.md)의 대상이다.
- 새 마크 추가, 마크 이동, 텍스트 마크 교정·삭제, 마크 삭제 도구의 삭제는 각각 실행 취소 한 단위다.
- 마크 이동의 경계와 동작은 [마크 이동](../mark-movement/spec.md)을 따른다.
- `⌘Z`는 마지막 이력을 취소하고 `⇧⌘Z`는 되살린다. 활성 도구는 바뀌지 않는다.
- 새 변경이 생기면 취소 후 남아 있던 다시 실행 이력은 폐기한다.
- 전체 지우기는 모든 마크와 진행 중 획을 비우는 1급 동작이다.

## 렌더링 품질

- Retina 화면에서도 모든 마크와 화살촉이 흐리지 않아야 한다.
- 빠른 포인터 이동에서도 경로가 눈에 띄게 끊기지 않아야 한다.
- 화면 녹화 중 입력 지연이 강의 흐름을 방해하지 않아야 한다.
- 그리기 모드를 나가거나 포인터 입력이 취소되면 진행 중 획, 빠른 추가 미리보기와 보정 미리보기를 남기지 않는다.

## 인수 조건

- 한 번의 포인터 제스처가 정확히 마크 하나를 만든다.
- 점 클릭, 짧은 경로, 빠른 경로가 모두 보이는 펜 마크로 확정된다.
- 빠른 추가로 다섯 결과를 만들 수 있고, 홀드 보정으로 화살표를 제외한 네 결과만 만들 수 있다.
- 확정된 직선·도형 마크는 이동·크기·기하 교정의 대상이 아니며 삭제와 Undo로만 바로잡는다.
- 마크 추가·교정·이동·삭제의 Undo/Redo가 원래 z-order를 보존한다.
- 실행 취소 뒤 새 마크를 만들면 이전 Redo가 다시 나타나지 않는다.
- 마크 삭제를 실행 취소해도 마크 삭제 도구가 유지된다.
- 전체 지우기와 통과 모드 전환 중 진행 중 획과 모든 미리보기가 남지 않는다.
- 다섯 색과 다섯 굵기가 커서, 플로팅 마커, 확정된 잉크 마크에서 일치한다.

## 비범위

- 부분 지우기와 드래그 연속 삭제
- 펜 마크의 경로 교정, 직선 마크의 끝점·화살촉 교정
- 직선·도형 마크의 이동·크기·기하·회전 교정과 여러 마크 동시 선택·이동
- 화살표 머리 인식과 자동 생성
- 화면 내용이나 다른 마크에 대한 스냅·정렬
