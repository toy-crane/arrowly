# 포인터 핑과 마크 삭제 보조 진입

## 목적

화면의 한 지점을 저장 없이 짧게 강조하고, 마크 삭제를 플로팅 마커와 키보드에서도 발견할 수 있게 한다.

그리기 도구의 선택·지속 상태, 이동·삭제 탐색, hit-test와 이력 계약은 [마크 도구 선택·이동·삭제 일관성](../mark-tool-consistency/spec.md)이 소유한다.

## 텍스트와 더블클릭

- 기본 텍스트 키 `T`로 텍스트 도구를 준비하고 원하는 위치를 한 번 클릭해 새 텍스트 편집 세션을 시작한다.
- 기존 텍스트 마크는 현재 도구와 관계없이 더블클릭하면 교정한다.
- 빈 곳 또는 텍스트가 아닌 마크를 더블클릭하면 해당 위치에 포인터 핑을 재생한다.
- 이동·삭제 탐색 또는 잠긴 삭제 도구가 포인터 입력을 소유하는 동안에는 더블클릭 핑을 만들지 않는다.
- 더블클릭의 첫 클릭이 만든 점 펜 마크는 두 번째 클릭에서 회수해 마크와 실행 취소 이력을 남기지 않는다.

## 포인터 핑

- 포인터 핑은 옐로우 `#FFD400` 마름모 파티클 8개가 입력점을 중심으로 8방향으로 퍼지는 버스트다.
- 파티클은 약 `200ms` 안에 반경 `34px`에 도달하고 그 위치에서 약 `300ms` 동안 잔상만 사라진다. 전체 수명은 `500ms`다.
- 입력 한 번마다 독립된 버스트 하나를 새로 만든다. 이전 버스트를 취소하거나 다시 시작하지 않는다.
- 빠른 연속 입력은 동시에 겹쳐 재생한다. 각 버스트는 이전 버스트와 경로가 구분되도록 방향을 조금씩 회전하고 반경을 소폭 달리한다.
- 포인터 핑은 hit testing을 가로막지 않으며 마크, 실행 취소 이력, 전체 지우기 대상과 저장 상태에 포함되지 않는다.
- 동작 줄이기 설정에서는 한 개의 정적 강조가 `150ms` 안에 사라지며 공간 이동을 사용하지 않는다.

## 마크 삭제 보조 진입

- `⌥`을 누른 동안의 순간 삭제 탐색이 기본 빠른 경로다.
- 플로팅 마커의 마크 삭제 버튼과 로컬 키 `E`는 같은 삭제 탐색 화면을 잠금 상태로 여는 보조 진입이다.
- 잠금 상태는 다른 도구를 고르거나 `E`를 다시 누를 때 끝난다.
- 텍스트 편집 중에는 글자 입력이 우선하고 `E`가 삭제 탐색을 시작하지 않는다.

## 인수 조건

- 텍스트 더블클릭은 교정을 시작하고 그 밖의 더블클릭은 포인터 핑을 만든다.
- 포인터 핑을 빠르게 세 번 입력하면 독립된 버스트 세 개가 겹쳐 보이고 각각 자신의 `500ms` 수명이 끝날 때 제거된다.
- 이동·삭제 제스처와 삭제 잠금 상태에서는 포인터 핑을 만들지 않는다.
- `E`와 마크 삭제 버튼이 같은 배경 딤·후보·hit-test를 사용하고 `E` 재입력으로 잠금을 끝낸다.

## 비범위

- 포인터 핑의 색·반경·속도 설정
- 지속적인 커서 하이라이트와 스포트라이트
- 빈 곳 더블클릭을 통한 새 텍스트 작성
