# 마크 이동

## 목적

강의 흐름을 멈추는 지속적인 선택 모드 없이, 모든 확정 마크의 잘못된 위치를 즉시 교정한다.

## 제품 계약

- 펜·텍스트·직선·화살표·도형을 포함한 모든 확정 마크를 이동할 수 있다.
- `⌘`을 `200ms` 유지하면 배경 딤과 모든 이동 후보를 표시한다.
- `200ms` 전에 직접 `⌘` 드래그하면 잡은 마크만 즉시 강조하고 지연 없이 이동 후보가 된다.
- 후보로 보이는 영역 전체가 실제 hit 영역이며, 겹친 곳에서는 가장 위의 마크 하나가 우선한다.
- `4px` 이하의 포인터 이동은 아무 변경도 만들지 않고 이를 넘은 드래그만 이동을 시작한다.
- 이동은 지속적인 선택 상태를 남기지 않고 현재 그리기 도구를 바꾸지 않는다.
- 표적화, 탐색 화면, 상태 전이, 이력과 전체 인수 조건의 단일 소스는 [마크 도구 선택·이동·삭제 일관성](../mark-tool-consistency/spec.md)이다.

## 비범위

- 여러 마크 동시 선택·이동, 그룹, 정렬과 스냅
- 지속적인 선택 상태, 이동 전용 도구와 선택 핸들
- 펜 경로, 도형 크기·회전, 꼭짓점·끝점 교정
