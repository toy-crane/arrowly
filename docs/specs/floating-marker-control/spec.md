# 플로팅 마커 도구·속성 위계

## 목적

플로팅 마커를 작게 유지하면서 자유곡선·텍스트·마크 삭제가 같은 도구 위계이고, 블랙보드는 독립된 배경 전환이며, 빠른 추가·색·굵기·텍스트 크기는 현재 도구의 속성이라는 관계를 한눈에 이해하게 한다.

통합된 최신 화면과 동작은 [빠른 마크 도구 프로토타입](../quick-mark-tools/prototype.html)이다. 기존 [도구·속성 위계 프로토타입](./prototype.html)은 빠른 마크 도구를 추가하기 전의 시각 기준으로만 남긴다.

## 1층 도구 위계

- 기본 캡슐의 1층에는 `자유곡선 도구`, `텍스트 도구`, `마크 삭제 도구`, 구분선, `블랙보드`를 순서대로 둔다.
- 자유곡선·텍스트·마크 삭제는 상호 배타적 도구다. 블랙보드는 배경 전환이므로 구분선 뒤에 격리하고 현재 도구와 독립적으로 켜고 끈다.
- 자유곡선 도구 아이콘은 도구 모양이 아닌 실제 출력인 열린 스윕을 사용한다. 넓은 두 호가 한 번만 방향을 바꾸며 작은 고리, 자기 교차, 추가 굴곡은 넣지 않는다.
- 자유곡선 아이콘에는 현재 잉크색과 현재 굵기의 상대 단계를 함께 적용한다.
- 빠른 추가가 준비된 동안 자유곡선 버튼은 선택한 사각형·타원·삼각형·직선·화살표 아이콘과 `1회` 상태를 나타낸다.
- 활성 도구와 활성 블랙보드는 중립색 배경으로 표시한다. 색은 현재 잉크색을 나타내는 용도로만 사용한다.
- 캡슐은 높이 `44px`, 셀 `42×32px`, 다크 반투명 재질을 유지한다.
- 캡슐은 좌우 외곽 `8px`, 자유곡선–텍스트와 텍스트–마크 삭제 사이 `4px`, 구분선 좌우 `8px`를 사용하며 전체 폭은 `209px`다.

## 2층 도구 속성

- 2층은 현재 도구에 연결된 하나의 맥락형 도구 속성 패널이다.
- 자유곡선 도구 패널은 세 줄이다.
  - 그리기 선택: `자유선`, `화살표`, `사각형`, `타원`, `삼각형`
  - 색: 옐로우·오렌지·핑크·그린·블루 5개
  - 굵기: 5단계
- 첫 줄의 다섯 도구 아이콘은 패널 위에서 검정이 아닌 중립 흰색으로 표시한다. 직선은 첫 줄에 두지 않고 홀드 보정으로만 만든다.
- 텍스트 도구 패널은 `24/32/44/60/80px`를 나타내는 비례형 `T` 표본 5개다. 첫 기본값은 `32px`다.
- 텍스트 크기의 숫자는 화면에 표시하지 않는다. 비례해서 커지는 `T` 표본으로 비교하고 정확한 `px` 값은 접근성 이름과 툴팁에만 제공한다.
- 접힌 텍스트 도구의 `T`는 현재 텍스트 크기를 반영한다. 실제 `24/32/44/60/80px`를 각각 `15/17/19/22/24px`로 압축해 표시한다.
- 패널에는 `빠른 추가·색·굵기·크기` 시각적 라벨을 표시하지 않는다. 패널과 각 선택지의 접근성 이름에는 속성 의미와 값을 유지한다.
- 패널은 고정 최소 폭 없이 콘텐츠 크기를 따르며 외곽 패딩 `8px`, 선택지 사이 `4px`, 행 사이 `4px`를 사용한다.
- 기본 위치는 캡슐 위다. 최상단 근처에서는 아래로 뒤집고, 좌우 경계에서는 화면의 `6px` 안전 여백 안으로 클램프한다. 연결 화살표는 실제 활성 도구 버튼의 중심을 가리킨다.
- 마크 삭제 도구에는 별도 속성 패널을 두지 않는다.

## 상호작용

- 비활성 도구 버튼을 누르면 해당 도구로 전환하고 패널은 접힌 상태를 유지한다.
- 이미 활성화된 자유곡선·텍스트 버튼을 다시 누르면 해당 속성 패널을 열거나 닫는다.
- 첫 줄의 자유선을 고르면 기본 자유곡선 도구로 복귀하고 패널을 닫는다. 나머지 네 도구를 고르면 패널을 닫고 해당 일회성 제스처를 준비하며, 마크 하나를 확정하면 자유곡선 도구로 복귀한다.
- 색·굵기·텍스트 크기를 고르면 즉시 적용하고 패널을 닫는다.
- 마크 삭제 버튼을 누르면 삭제 도구로 전환한다. 마크 하나를 지워도 활성 상태를 유지하며 다른 도구를 눌러야 종료된다.
- 바깥 누르기, 그리기 시작, 블랙보드 전환, 키보드 도구 전환은 열린 패널을 닫는다.
- 키보드 텍스트 키 `T`는 텍스트 도구를 준비하되 속성 패널은 열지 않는다.
- 키보드 삭제 키 `E`는 마크 삭제 도구를 활성화하되 속성 패널을 열지 않는다.
- 텍스트 편집 중에도 텍스트 버튼을 눌러 크기 패널을 열 수 있다. 크기를 고르면 편집 세션은 유지하고 패널만 닫는다.
- 자유곡선 또는 마크 삭제 버튼을 누르면 진행 중 텍스트 편집을 기존 확정 규칙에 따라 끝낸다.
- 블랙보드를 켜고 꺼도 현재 도구는 유지한다.
- 캡슐 드래그, `4px` 탭·드래그 임계값, 위치 저장은 기존 동작을 유지한다.

## 상태

- `펜 기본`: 자유곡선 도구 활성, 패널 닫힘
- `펜 속성`: 빠른 추가·색·굵기 패널 열림
- `빠른 추가`: 선택한 기하 아이콘과 `1회` 상태, 패널 닫힘
- `텍스트 기본`: 텍스트 도구 활성, 패널 닫힘
- `텍스트 속성`: 다음 텍스트의 기본 크기 패널 열림
- `텍스트 편집`: 기존 또는 새 텍스트 편집 세션과 현재 텍스트 크기 표시
- `마크 삭제`: 삭제 도구 활성, 패널 없음
- `보드`: 현재 도구를 유지한 채 블랙보드 활성
- `경계 배치`: 상단 방향 반전 또는 좌우 경계 보정이 적용된 상태

## 인수 조건

- 접힌 캡슐에서 자유곡선·텍스트·마크 삭제가 같은 위계로 보이고 블랙보드는 구분선 뒤에 보인다.
- 캡슐 높이는 `44px`, 전체 폭은 `209px`이며 셀과 간격이 확정 규격을 따른다.
- 자유곡선 아이콘만 보고 현재 잉크색과 상대 굵기를 알 수 있고 모든 굵기에서 열린 스윕 실루엣을 유지한다.
- 자유곡선 패널에는 `자유선 → 화살표 → 사각형 → 타원 → 삼각형` 순서의 그리기 선택·색·굵기만, 텍스트 패널에는 텍스트 크기만 나타나며 마크 삭제에는 패널이 없다.
- 패널에는 시각적 라벨이 없지만 패널·행·선택지의 접근성 이름으로 의미를 구분할 수 있다.
- 빠른 추가 선택과 1회 확정 뒤의 자유곡선 복귀가 도구 아이콘과 활성 상태에 즉시 반영된다.
- 마크 삭제 도구는 여러 번 삭제해도 활성 상태를 유지한다.
- 텍스트 편집 중 크기를 바꿔도 편집 세션이 유지된다.
- 최상단·좌우 가장자리에서도 패널이 화면 밖으로 나가지 않고 연결 화살표가 활성 도구를 가리킨다.
- 실제 macOS 오버레이에서 마우스로 캡슐을 옮기고 모든 도구·속성을 선택할 수 있다.

## 비범위

- 선택 도구, 도형 편집 핸들, 부분 지우개
- 현재 굵기를 숫자·보조 배지·두 번째 아이콘으로 중복 노출
- 속성 패널 중첩 또는 3단 메뉴
- 캡슐 높이·드래그·저장 방식 변경

## 남은 위험

- 넓어진 `209px` 캡슐과 3줄 자유곡선 패널은 작은 화면과 실제 네이티브 캡슐에서 시각 회귀 확인이 필요하다.
- non-activating `NSPanel`에서는 합성 포인터 이벤트가 실제 클릭을 대체하지 못할 수 있으므로 구현 후 네이티브 수동 검증이 필요하다.
